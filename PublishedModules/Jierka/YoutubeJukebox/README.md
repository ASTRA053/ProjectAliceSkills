# YoutubeJukebox

### Download

##### > WGET method
```bash
wget http://modules.projectalice.ch/YoutubeJukebox \
-O ~/ProjectAlice/system/moduleInstallTickets/YoutubeJukebox.install
```

##### > Alice CLI method
```bash
alice module:install Jierka/YoutubeJukebox
```
